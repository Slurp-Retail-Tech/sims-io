  - title: "Franchise import failed with 'Unauthenticated' (401)"
    detail: "The default POS import URL used http://, so requests were redirected to https:// and the Authorization header was stripped during the redirect — causing the POS API to reject every import with 'Unauthenticated.'. The default URL (and the .env.example entry) now use https:// so the auth header is preserved and the import authenticates correctly. If your environment sets POS_IMPORT_URL explicitly, make sure it uses https://."
