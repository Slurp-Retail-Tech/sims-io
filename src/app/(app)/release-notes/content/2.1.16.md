fixed:
  - title: "Franchise import failed mid-run with 'Unauthenticated' on later pages"
    detail: "The import logs in once and then pages through every franchise, but a large import can take long enough that the POS token expires before the last page — causing later pages to fail with 'Unauthenticated' (401) even though earlier pages succeeded. The import now re-authenticates once and retries the affected page when it hits a 401, so the run continues to completion. A page that still fails after a fresh login is reported as a genuine auth error."
