# Triage Bug

## Workflow

1. Normalize the report into environment, steps, expected result, and actual result.
2. Identify missing information and likely owning area.
3. Estimate severity, blast radius, and reproducibility.
4. Suggest the best next subagent or workflow.
5. Produce a clean issue summary with next actions.

## Output

- normalized bug summary
- severity
- reproducibility
- suspected area
- missing information
- next action
